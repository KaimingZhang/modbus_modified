mc_free(3)
==============


NAME
----
mc_free - free a libmc context


SYNOPSIS
--------
*void mc_free(mc_t *'ctx');*


DESCRIPTION
-----------
The _mc_free()_ function shall free an allocated mc_t structure.


RETURN VALUE
------------
There is no return values.
